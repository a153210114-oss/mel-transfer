# 华伴平台免责与资质核验条款 1.0

> 本文件为产品运营条款草案，正式上线前建议由当地律师按目标国家/地区复核。

## 一句话原则

华伴不做中间商，不赚差价；华伴提供信息连接、AI 工具、信用记录和资质核验辅助，让需要的人遇见可靠的人。

## 平台角色

1. 华伴是信息连接与工具平台，不是供应商、代理商、雇主、旅行社、运输公司、法律/会计/移民服务机构。
2. 用户与供应商之间的商品或服务交易，由双方直接沟通、确认共识合约并完成。
3. 华伴可提供需求整理、服务者展示、联系方式交换、AI 客服、信用评价、投诉记录和公开资料核验辅助。

## 供应商责任

供应商提交申请或发布服务/商品时，应确认：

1. 提交的姓名、联系方式、服务范围、商品信息、价格说明、资质证明和公开链接真实有效。
2. 如其服务依法需要政府许可、行业资质、保险、执照或注册，应自行取得并保持有效。
3. 供应商自行承担履约、商品质量、服务质量、安全、税务、消费者权益、隐私保护和当地法律合规责任。
4. 供应商不得冒用他人资质、虚假宣传、误导用户，或提供违法违规服务。

## 资质核验方式

1. AI agent 可先向供应商收集资质编号、注册名称、执照名称、政府公开查询网址或行业协会公开页面。
2. AI agent 可根据国家、城市和服务类型，提示应补充的资质字段，并整理公开核验路径。
3. 对政府公开网站或行业公开页面可查的资质，平台记录公开查询结果、链接、查询时间和关键信息。
4. AI 核验结果仅作为管理员审核辅助，不等同于官方认证、法律意见或平台担保。
5. 管理员人工复核后，才可开放供应商的供给中心入口。

## 平台免责

1. 华伴不保证供应商一定适合用户需求，也不保证商品或服务结果满足用户预期。
2. 华伴不对供应商的实际履约、服务质量、商品质量、安全事故、延误、取消、纠纷、损失或违法行为承担替代责任。
3. 华伴展示“资料已提交”“资质已核验”时，仅表示平台基于供应商提交的信息和公开渠道完成了有限核验，不表示政府背书或平台担保。
4. 用户在交易前应自行判断服务者资质、价格、服务范围、取消政策和风险。
5. 对涉及法律、会计、移民、医疗、金融、保险等专业服务的信息，用户应自行确认服务者当地执业资格，并独立决定是否委托。

## 平台可采取的措施

华伴发现以下情况时，可要求补充资料、限制展示、暂停供给中心权限或移除供应商：

- 资质过期、无法核验或与服务范围不匹配
- 用户投诉较多或存在安全风险
- 虚假宣传、冒用资质、恶意引流或私下欺诈
- 违反当地法律法规或平台规则
- 不配合纠纷协调或信用评价
