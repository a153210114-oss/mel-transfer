# PG_01 华伴推广 Agent 开工手册

目标：让推广 Agent 按三类人群精准生成任务、文案和素材，让人工只做最后确认、发布和反馈记录。

## 三条推广线

### 1. 年轻海外华人 / 留学生

核心情绪：

- 不想被算法安排人生。
- 不想被红点、群聊、广告和推荐流拖走。
- 想静静地做自己想做的事。

主文案：

> 不想被红点、广告和推荐流拖着走？
> 华伴 AI+，你主动，它才出现。
> 一个安静、有用、听你指挥的 AI 工具。

投放入口：

- 小红书：留学、海外生活、独居、AI 工具、效率工具、反内耗。
- 留学生群、学校群、租房群。
- TikTok / Reels 短视频。

落地参数：

- `https://www.huabanapp.com/?ref=student_free`

Agent 首句方向：

> 哈哈，是留学生吗？想让我陪你学习、生活，还是搞点 AI 创意？

### 2. 家庭用户 / 爸爸妈妈

核心情绪：

- 不想被时代落下。
- 希望全家都能轻松用 AI。
- 希望简单、安全、少打扰。

主文案：

> 全家用 AI，就用华伴 AI+。
> 爸爸妈妈宝宝都能用。
> 一个有脑、有手、有温度的家庭 AI 工具。

投放入口：

- 华人妈妈群、家长群、社区群。
- 微信朋友圈。
- 小红书：海外育儿、陪读、家庭生活、父母用 AI。

落地参数：

- `https://www.huabanapp.com/?ref=family_ai`

Agent 首句方向：

> 哈哈，这是谁呀？爸爸妈妈还是宝宝呀？

### 3. 事业 / 小生意 / 手艺人

核心情绪：

- 让更多需求找到自己。
- 让能力、产品和服务被看懂、被信任、被联系。
- 不想刷广告，也想让事业成型并推向更大的市场。

主文案：

> 让更多需求找到你。
> 华伴能当你的事业助手。
> 帮你整理业务、打磨介绍、生成推广内容、记录线索，让事业成型，并推向更大的市场。

投放入口：

- 华人司机群、维修群、餐馆群、配送群、商家群。
- 今日墨尔本、Yeeyi、Go2AU、Facebook 华人工作/服务群。
- 小红书：副业、接单、海外小生意、华人服务。

落地参数：

- `https://www.huabanapp.com/?ref=supplier_income`

Agent 首句方向：

> 你现在做什么事业？在哪个城市？我先帮你整理成别人看得懂、愿意联系的介绍。

## 今日实施版

今天先不追求铺太广，只追求真实反馈。

### 今日发布顺序

1. 家庭用户：朋友圈或 1 个家长/妈妈群。
2. 年轻/留学生：小红书或 1 个留学生/租房群。
3. 事业/小生意：1 个司机/餐饮/维修/服务者群。

### 今天只看 5 个结果

- 有没有人点开链接。
- 有没有人领养华伴 AI。
- 有没有人问“这个怎么用”。
- 有没有服务者说“我也可以提供服务/商品”。
- 有没有人反感、看不懂或觉得太虚。

### 今天人工记录

- 发布入口：发到哪里。
- 发布文案：发了哪一版。
- 咨询数量：几个人问。
- 领养/试用：几个人点开。
- 反馈原话：好的、不懂的、反感的都记录。

## 推广 Agent 分工

### Agent 自动做

- 从中文入口库挑选当天适合的入口。
- 判断入口对应人群：年轻/留学生、家庭、小生意/手艺人。
- 生成对应文案、海报文字、二维码链接和短视频脚本。
- 生成后台推广任务。
- 标记发布等级：
  - A：自有或授权入口，可自动发布。
  - B：人工确认后发布。
  - C：只观察，不推广。
- 记录点击、领养、分享、咨询、服务者申请和反馈。
- 根据效果给出第二天任务建议。

### Agent 不做

- 不自动注册第三方账号。
- 不自动加微信群、Facebook 群、论坛。
- 不自动群发私信。
- 不刷屏。
- 不绕过验证码、审核或平台限制。
- 不假装普通用户推荐华伴。

### 人工运营做

- 查看后台 `推广任务`。
- 确认入口能不能发。
- 微调 Agent 文案，让它更像真人。
- 人工发布到允许的群、平台、朋友圈或小红书。
- 回复咨询。
- 把截图、咨询量、领养量、反馈录回后台。

### 你亲自做

- 选择每天主攻哪条线。
- 决定哪些群或平台可以发。
- 发送第一批关键群/朋友圈。
- 看真实用户反应，告诉华伴哪些文案打动人、哪些让人反感。
- 审核重要合作者、服务者和合伙人线索。

## 每日 60 分钟启动版

1. 5 分钟：选今天主攻线。
2. 10 分钟：Agent 生成 3 条任务。
3. 15 分钟：你挑 1 条文案微调。
4. 15 分钟：人工发布 1-2 个入口。
5. 10 分钟：回复咨询或记录反馈。
6. 5 分钟：标记效果，好/一般/差。

最低目标：

- 每天 1 条线。
- 每天 1 张海报或 1 条短视频脚本。
- 每天 2 个入口发布。
- 每天记录 3 条真实反馈。

## 后台任务字段建议

- `segment`：`student_free` / `family_ai` / `supplier_income`
- `source_name`：入口名称
- `source_url`：入口链接
- `copy_short`：短文案
- `copy_long`：长文案
- `landing_url`：带 ref 的落地页
- `channel_level`：A/B/C
- `daily_target`：发布、回复、反馈目标
- `metrics`：点击、领养、咨询、申请、分享
- `compliance_notes`：合规边界

## 第一周目标

- 每条线至少投放 5 个入口。
- 每条线至少收集 10 条真实反馈。
- 找出点击率最高的 3 句文案。
- 找出最愿意分享的人群。
- 找出最容易留下资料的服务者类型。

判断标准：

- 有人主动问：有效。
- 有人扫码领养：有效。
- 有人分享：强有效。
- 有人留下服务/城市/联系方式：高价值。
